# Data

This folder contains processed datasets used for Lake Urmia environmental analysis.

## Included Datasets

- Surface water area estimates (2016–2025)
- Climate variables
- Drought indicators (SPEI)
- Environmental predictors for machine learning models

## Data Sources

The analysis integrates publicly available satellite and climate datasets, including:

- Sentinel-2 satellite imagery
- CHIRPS precipitation data
- ERA5-Land temperature data
- MOD16 evapotranspiration data

Original satellite and climate datasets were obtained from publicly available sources.
